Detailed Description

The QMailMessageThreadedModel class provides access to a tree of stored messages.

The QMailMessageThreadedModel presents a tree of all the messages currently stored in the message store. By using the setKey() and sortKey() functions it is possible to have the model represent specific user filtered subsets of messages sorted in a particular order.

The QMailMessageThreadedModel represents the hierarchical links between messages implied by conversation threads. The model presents messages as children of predecessor messages, where the parent is the nearest ancestor of the message that is present in the filtered set of messages.

By default, the model will not match any messages, display messages in the order they were submitted, and mail store updates are not ignored.
