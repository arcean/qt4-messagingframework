Detailed Description

The QMailServiceConfiguration class provides a simple framework for creating wrappers classes that simplify service configuration management.

QMailServiceConfiguration provides a simple interface for manipulating the configuration parameters of a single service within an account configuration. For each specific service implemented, a configuration class derived from QMailServiceConfiguration should be implemented to make the configuration easily accessible.

See also QMailAccountConfiguration::ServiceConfiguration.

Member Type Documentation

enum QMailServiceConfiguration::ServiceType

This enum type is used to describe the type of a service

ConstantValueDescription
QMailServiceConfiguration::Unknown0The type of the service is unknown.
QMailServiceConfiguration::Source1The service is a message source.
QMailServiceConfiguration::Sink2The service is a message sink.
QMailServiceConfiguration::SourceAndSink3The service is both a message source and a message sink.
QMailServiceConfiguration::Storage4The service is a content manager.

See also QMailMessageSource and QMailMessageSink.
