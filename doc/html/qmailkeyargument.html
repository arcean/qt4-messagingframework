Detailed Description

The QMailKeyArgument class template defines a class representing a single criterion to be applied when filtering the QMailStore constent with a key object.

A QMailKeyArgument<PropertyType, ComparatorType> is composed of a property indicator, a comparison operator and a value or set of values to compare with. The type of the property indicator depends on the type that is to be filtered.
