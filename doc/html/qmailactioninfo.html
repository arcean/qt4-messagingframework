Detailed Description

The QMailActionInfo class provides the interface for tracking individual actions

QMailActionInfo provides the mechanism for messaging clients to track an action that is or has run on the server. QMailActionInfo objects can only be created by QMailActionObserver and derive most of their functionality and signals from QMailServiceAction.

QMailActionInfo presents requestType() and id() to the the client. For future QML interoperability properties and notification signals have been added.
