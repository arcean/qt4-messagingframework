Detailed Description

The QMailMessageServicePluginInterface class defines the interface to plugins that provide messaging services.

The QMailMessageServicePluginInterface class defines the interface to message service plugins. Plugins will typically inherit from QMailMessageServicePlugin rather than this class.

See also QMailMessageServicePlugin, QMailMessageService, and QMailMessageServiceFactory.
