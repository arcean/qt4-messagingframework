Detailed Description

The QMailThread class represents a thread of mail messages in the mail store.

QMailThread represents a thread (also known as conversation) of mail messages.

See also QMailMessage and QMailStore::thread().

Member Function Documentation

QMailThread::QMailThread ()

Constructor that creates an empty and invalid QMailThread. An empty thread is one which has no id or messages account. An invalid thread does not exist in the database and has an invalid id.

QMailThread::QMailThread ( const QMailThreadId & id )

Constructor that creates a QMailThread by loading the data from the message store as specified by the QMailThreadId id. If the thread does not exist in the message store, then this constructor will create an empty and invalid QMailThread.
