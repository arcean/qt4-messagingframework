Detailed Description

The QMailMessageSink class defines the interface to objects that provide external message transmission services to the messageserver.

The Qt Extended messageserver uses the QMailMessageSink interface to cooperate with components loaded from plugin libraries, that act as external transmitters of messaging data for the messaging framework. Instances of QMailMessageSink are not created directly by the messageserver, but are exported by QMailMessageService objects via their sink function.

See also QMailMessageService.

bool QMailMessageSink::transmitMessages ( const QMailMessageIdList & ids )   [virtual slot]

Invoked by the message server to initiate a transmission operation.

Attempt to transmit each message listed in ids to the external server.

Successfully transmitted messages should be progressively reported via messagesTransmitted().

Messages for which for which an unsuccessful attempt to transmit has been made should be progressively reported via messagesFailedTransmission().

Return true if an operation is initiated.
