Detailed Description

The QMailMessageServiceEditor class provides an interface that allows a service to be edited graphically.

QMailMessageServiceEditor provides the base class for a GUI component that can edit the configuration for a messaging service.

Member Function Documentation

void QMailMessageServiceEditor::displayConfiguration ( const QMailAccount & account, const QMailAccountConfiguration & config )   [pure virtual]

Invoked to set the editor with the details of the account account, described by config.

bool QMailMessageServiceEditor::updateAccount ( QMailAccount * account, QMailAccountConfiguration * config )   [pure virtual]

Invoked to update the account account and configuration config with the details currently displayed by the editor. Return true if the account and configuration are appropriately updated, and any necessary data storage external to the mail store has been performed.
