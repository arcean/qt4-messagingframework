Overview

The Messaging Framework Message Server Support Library provides classes that assist with the development of components that operate as plugins within the Message Server daemon. The client utility library depends on the Messaging Framework Client Library to provide data access and manipulation classes.

For detailed class information refer to: Message Server Support Classes.

Message Accounts

The Message Server Support Library provides classes for storing information about messaging accounts both outgoing and incoming.

ClassDescription
QMailServiceConfigurationProvides an interface to the configuration details for a service within an account.
QMailStoreAccountFilterForward the signals of QMailStore, filtering for those that affect a single account.

Message Services

The library provides classes that assist with the development of services that can be loaded into the {MessageServer Daemon}{Message Server} daemon via plugins.

ClassDescription
QMailMessageServiceThe interface between the message server and components loaded from plugin libraries.
QMailMessageSinkThe interface to objects that provide message external transmission services to the messageserver.
QMailMessageSourceThe interface to objects that provide access to externally sourced messages to the messageserver.
QMailMessageServiceEditorInterface that allows a service to be edited graphically.
QMailMessageServiceConfiguratorInterface that allows a service to be configured.
QMailMessageServiceFactoryCreates objects implementing the QMailMessageService interface.

Utility Classes

The library also provides some utility classes that are used by service components within the Message Server daemon.

ClassDescription
QMailTransportA class that provides an encrypted line-oriented communications channel.
QMailMessageClassifierA class that classifies message content by inspection.
