Detailed Description

The QMailContentManagerPluginInterface class defines the interface to plugins that provide message content management facilities.

The QMailContentManagerPluginInterface class defines the interface to message content manager plugins. Plugins will typically inherit from QMailContentManagerPlugin rather than this class.

See also QMailContentManagerPlugin, QMailContentManager, and QMailContentManagerFactory.
